WHAT IS BLACK STACK?

Black Stack is a display website for the learning and testing of web development concepts by Thapelo Sepale. Thapelo is an aspiring programmer with a keen interest in Php and MySql web development. Thapelo, who is also known by the name "JT31WEB" on the internet, is a junior web developer from Klerksdorp in South Africa.


JT31WEB has learnt the basics of HTML5, CSS3, JAVASCRIPT, PHP & MYSQL from various E-books, YouTube videos and websites. He is also familiar with other software development concepts such as user interface design, web services, basic web security and SDLC. Thapelo is dedicated to learning the latest web technology and is now regularly updating his Github repository to both get more familiar with the Git version control system and to display and get input from better developers


Thapelo is currently working on the various aspects of being a FULL STACK DEVELOPER and hopes to one day teach these skills to less fortunate kids & adults who never had an opportunity to learn them. He is also very passionate about education and self improvement and believes that the internet is by far the best tool to offer the most trusted and innovative educational content and tools. He also plans to convert the current BlackStack website into a educational platform where people can access and discuss various web development concepts
